# Prompt-to-Voice

## Overview
**Prompt-to-Voice** is a web tool that converts typed text prompts into speech using JavaScript.  
It demonstrates basic text-to-speech functionality in the browser â€” perfect for showcasing interactive JavaScript skills.

This project uses the built-in Web Speech API to generate spoken audio from text input.

## Features
- Converts text to voice in real time  
- Simple and clean user interface  
- Works directly in the browser (no backend needed)  
- Uses Web Speech API
